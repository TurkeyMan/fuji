MF_API uint32 MFImage_GetPlatformAvailability ( int  format)

Gets a variable representing which platforms support the specified image format in hardware.

Parameters
formatFormat to test for hardware support.
Returns
Result is a bitfield where each bit represents hardware support for a specific platform. Platform support can be tested, for example, using: ( result & MFBIT(FP_PC) ) != 0.
